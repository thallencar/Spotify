## 🐞 Resolvendo Bugs
 
Segue o passo a passo, caso projeto não carregue as imagens e/ou nome dos artistas.

```bash
npm uninstall -g json-server

npm install -g json-server@0.17.4

json-server --watch api-artists/artists.json --port 3000
```
